# iFrame Sample App

This app is available to all Zendesk accounts via GitHub. Please submit bug reports to [Zendesk](https://support.zendesk.com/requests/new). Pull requests are welcome.

### Short Description

This is the iFrame Sample App for the new Zendesk app marketplace that is available at all app locations in Lotus, including nav_bar, top_bar, ticket_sidebar, new_ticket_sidebar, and user_sidebar. See [Manifest](http://developer.zendesk.com/documentation/apps/manifest.html) for more details about locations.

This app aims to be a good start to learn how to iframe things of different sizes and methods.

### Feature List

1. This app displays the [Zendesk Apps Introduction page](http://developer.zendesk.com/documentation/apps/introduction.html) in an iFrame.

2. This app also displays the [Zendesk Support page](http://support.zendesk.com) in an iFrame.

3. The Zendesk Apps Introduction page is available when the app is activated in top_bar and nav_bar.

4. The Zendesk Support page is available when the app is activated in user_sidebar, ticket_sidebar, and new_ticket_sidebar.
